# Student Registration Form

## Description
A responsive student registration form built using HTML, CSS and JavaScript. This project includes a clean UI, image upload preview feature and mobile-friendly layout.

## Features
- Responsive design (mobile, tablet, desktop)
- Image upload with preview
- Clean and modern UI
- Form validation basics

## Technologies Used
- HTML
- CSS
- JavaScript
